# Intro to Functions

## Function Syntaxs
- Function Declaration
```javascript
    function name(parameter1, parameter2, parameter3) {
  // body
}
```
- Function Expressions
```javascript
    const name = function (params) {
        body
    }
```
- Arrow Functions
```javascript
    const name = (params) => {
        body
    }
```
